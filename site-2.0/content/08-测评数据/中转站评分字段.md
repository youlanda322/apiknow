# 中转站评分数据字段

## 基础字段

- `id`：稳定英文标识。
- `name`：站点公开名称。
- `operator`：运营主体及核验方式。
- `official_url`：官网。
- `updated_at`：最后核验日期。
- `evidence_status`：已核验 / 证据待补 / 价格变动 / 已下架。
- `commercial_relation`：无 / 联盟链接 / 赞助 / 获得测试额度。

## 评分字段

- `reliability`：0-25。
- `pricing`：0-20。
- `models`：0-15。
- `compatibility`：0-15。
- `security`：0-15。
- `support`：0-10。
- `total`：六项相加，满分 100。
- `grade`：S / A / B / C / 观察。

## 测试字段

- 测试地区、网络与出口。
- 测试日期、时段与持续时间。
- 模型、版本、参数和接口。
- 样本量、成功定义和失败定义。
- P50、P95、P99、成功率、429、5xx、超时。
- 原始响应、请求 ID、测试脚本和数据文件路径。

## 风险字段

- 风险类型、事实描述、证据、发现日期、影响范围和复核状态。
- 硬性规则命中时，不计算常规推荐等级。
- 商业关系不能影响风险字段和分项得分。
