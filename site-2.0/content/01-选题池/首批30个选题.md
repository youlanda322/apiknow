# 首批 30 个深度选题

## API 基础

1. 一次 API 请求经历了什么：DNS、TLS、网关与业务服务。
2. HTTP 状态码不够用：怎样设计稳定的业务错误码。
3. API Key、OAuth 和请求签名分别解决什么问题。
4. REST 分页的四种方式：offset、cursor、keyset 与时间窗口。
5. API 版本控制：URL、请求头和兼容性策略怎么选。

## 模型与接口

6. 大模型 API 成本模型：Token、缓存、重试与长上下文。
7. 流式 LLM API：SSE 事件、首 Token 和中断恢复。
8. 工具调用不是 JSON 输出：状态机与参数校验。
9. Embedding API 选型：维度、距离、切块与迁移成本。
10. 图像生成 API 上生产：异步任务、回调和对象存储。

## 实战开发

11. 重试为什么造成重复扣费：幂等键与退避。
12. Webhook 安全：签名、重放保护、去重和补偿。
13. Node.js 接入第三方 API 的超时与连接池配置。
14. Python API 客户端怎样做类型、错误和重试封装。
15. 在多供应商之间实现可回滚的适配层。

## 测评对比

16. 选 API 不只看单价：七维评估框架。
17. 大模型横向测评怎样避免提示词和样本偏差。
18. 单位有效结果成本：把质量返工纳入价格比较。
19. 官方 API 与中转 API：风险边界和适用条件。
20. 开源自托管与托管 API：总拥有成本怎么计算。

## 故障排查

21. 429 到底是谁限流：客户端、网关还是上游模型。
22. 偶发 502 的分层排查：DNS、连接池、代理和上游。
23. 平均延迟正常，用户为什么仍觉得慢：P95/P99。
24. 流式响应本地正常，上线后卡顿：代理缓冲排查。
25. TLS 握手失败：证书链、SNI、时间和协议版本。

## 行业观察

26. 模型版本频繁变化时怎样做回归测试和灰度。
27. OpenAI 兼容接口的“兼容”究竟到哪一层。
28. API 平台的退出成本：代码、数据、提示词和评估集。
29. AI API 数据保留条款应该怎样读。
30. 从计费单位变化看模型服务的产品策略。
